# Thai Pharma Image Prompt Builder

A standalone HTML app for creating compliance-aware image prompts for pharmacy services, healthcare communication, HCP education, patient support, and pharmaceutical marketing in Thailand.

Created and curated by **ดร.ภก.วิรุณ เวชศิริ** for **Pharm Connection (Thailand) Co., Ltd.**

## Features

- Pharm Connection CI styling: Navy `#1B2F72`, Royal Blue `#2563B5`, Cyan `#00AEEF`, Ice White `#F0F8FD`
- Prompt font via Google Fonts
- Pharm Connection logo via CDN
- Prompt templates for:
  - Pharma Product Hero
  - Pharmacy Service Poster
  - 9-Panel Storyboard
  - HCP Sales Aid / Medical Education Visual
  - Patient Support Program UI Mockup
  - Medical Congress Booth Key Visual
  - Comparison Infographic
  - Brand Identity Board
  - Professional Pharmacist / MSL Portrait
  - Miniature Medication Safety Diorama
- Built-in compliance guardrails for healthcare and pharmaceutical visual communication
- Copy prompt and download `.txt` functions

## Usage

Open `index.html` in a browser, select key elements, and generate an image prompt.

## Notes

This tool supports prompt drafting and visual planning. It is not a regulatory, legal, medical, or advertising approval tool. Review all output through appropriate medical, regulatory, legal, and brand governance processes before real-world use.
